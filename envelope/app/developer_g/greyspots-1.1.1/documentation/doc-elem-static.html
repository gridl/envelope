Static

Tag:

<gs-static></gs-static>

Description:

The <gs-static> allows you to view data from a <gs-form> without the ability to change that data.

Examples:

Skeleton Example:

A basic <gs-static> control will have the following behaviors:

  1. The [value=""] attribute will set the text of the control.
  2. The control cannot be focused.
  3. The text can be selected so that it can be copied.
Attribute [column]:

When you have a gs-static inside a data element and you want to use it to display cell data you use the column attribute.

Attribute [pre]:

Use the [pre] attribute to format the text. Note that a \n in the [value=""] attribute will show as \n, but a newline in the attribute text will show properly.

Attribute [qs=""]:

The [qs=""] attribute can be used to sync the <gs-static> with the query string.

Attribute [inline]:

Sometimes you are limited on space. The [inline] attribute can be used to make the <gs-static> smaller width-wise.

Attribute [mini]:

Sometimes you are limited on space. The [mini] attribute can be used to make the <gs-static> smaller height-wise.
