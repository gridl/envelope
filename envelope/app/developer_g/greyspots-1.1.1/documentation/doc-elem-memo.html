Memo

Tag:

<gs-memo></gs-memo>

Description:

The purpose of a <gs-memo> control is to hold large amounts of text from the user.

Examples:

Skeleton Example:

A basic <gs-memo> element. Without attributes the <gs-memo> control will have the following behaviors:

  1. You can resize the <gs-memo>.
  2. A scrollbar will appear when the text is longer then the <gs-memo>.
Attribute [onchange=""]:

To run javascript when the value changes, use the [onchange=""] attribute.

Attribute [value=""]:

To set the default value for the <gs-memo>, use the [value=""] attribute.

Attribute [qs=""]:

The [qs=""] attribute can be used to sync the <gs-memo> with the query string.

Attribute [placeholder=""]:

The [placeholder=""] attribute can be used when you don't want to use a label to explain the purpose of the control to the user. (The label and the placeholder can be used together, that's just not what normally happens.)

Attribute [autoresize]:

Rather then manually resizing the <gs-memo> when the text gets too large, it can automatically resize itself when you use the [autoresize] attribute.

Attribute [no-resize-handle]:

When you use the [no-resize-handle] attribute, you can no longer resize the memo. When you use both the [autoresize] attribute, you typically also want [no-resize-handle].

Attribute [mini]:

Sometimes you are limited on space. The [mini] attribute can be used to make the <gs-memo> smaller.

Attribute [tabindex=""]:

With the [tabindex=""] attribute, you can control what order your controls are tabbed to.

Attribute [disabled]:

With the [disabled] attribute, you can control whether or not you can use the control.
