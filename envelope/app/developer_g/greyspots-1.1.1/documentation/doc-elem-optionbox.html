Optionbox

Tag:

<gs-optionbox>
    <gs-option>...</gs-option>
</gs-optionbox>

Description:

A <gs-optionbox> control allows you to select an option from a list.

Examples:

Skeleton Example:

A basic <gs-optionbox> element with six options. Without attributes the optionbox will have the following behaviors when an option is clicked:

  1. Select the clicked option.
  2. Make the clicked option yellow.
  3. Put a dot in the icon of the option.
  4. Deselect any other selected options in the same box.
Attribute [onchange=""]:

To run javascript when the value changes, use the [onchange=""] attribute.

Attribute [no-target]:

The "no-target" attribute prevents the default styling of the options.

Horizontal optionbox:

To run make the optionbox horizontal, put the [inline] attribute on the gs-options.

Attribute [mini]:

The "mini" attribute removes the border from the gs-optionbox.

Option Attribute [value=""]:

When you get the value of a <gs-optionbox>, it will use the [value=""] attribute of the <gs-option> to return. If there is no [value=""] attribute on the <gs-option>, it will return the text of the <gs-option>.

Optionbox Attribute [value=""]:

To set the default value for the <gs-optionbox>, use the [value=""] attribute.

Attribute [qs=""]:

The [qs=""] attribute can be used to sync the <gs-optionbox> with the query string.

Attribute [disabled]:

With the [disabled] attribute, you can control whether or not you can use the control.
