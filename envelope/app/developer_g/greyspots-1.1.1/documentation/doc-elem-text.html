Description:

Examples:

Skeleton Example:

Without attributes the <gs-text> will have the following behaviors:

  1. Clicking on the control gives you a cursor.
  2. You can change the text by typing.
Attribute [onchange=""]:

To run javascript when the value changes, use the [onchange=""] attribute.

Attribute [value=""]:

To set the default value for the <gs-text>, use the [value=""] attribute.

Attribute [qs=""]:

The [qs=""] attribute can be used to sync the <gs-text> with the query string.

Attribute [placeholder]:

The [placeholder] attribute can be used when you don't want to use a label to explain the purpose of the control to the user. (The label and the placeholder can be used together, that's just not what normally happens.)

Attribute [mini]:

Sometimes you are limited on space. The [mini] attribute can be used to make the <gs-text> smaller.

Attribute [tabindex=""]:

With the [tabindex=""] attribute, you can control what order your controls are tabbed to.

Attribute [autocorrect=""]/[autocapitalize=""]/[autocomplete=""] Attribute:

If you don't want spelling to autocorrect, you can use the html5 [autocorrect="off"] attribute. See also html5 [autocapitalize="off"] and [autocomplete="off"].

Attribute [disabled]:

With the [disabled] attribute, you can control whether or not you can use the control.
